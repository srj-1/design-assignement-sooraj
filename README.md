# Digital IC Design Internship

This repository contains the assignments and projects completed during my Digital IC Design Internship. The designs are implemented using Verilog HDL and verified through simulation using dedicated testbenches. The repository covers fundamental combinational and sequential digital circuits, providing hands-on experience in RTL design, verification, and digital system development.
