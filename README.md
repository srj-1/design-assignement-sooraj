Digital IC Design Internship

This repository contains the RTL design and verification assignments completed during my Digital IC Design Internship. The projects are implemented using Verilog HDL and verified through simulation using Vivado. The assignments cover both combinational and sequential digital circuits, helping build a strong foundation in RTL design, testbench development, and functional verification.
